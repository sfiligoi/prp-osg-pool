Note: Relies on
https://github.com/sfiligoi/prp-htcondor-portal

For each supported namespace:
1) deploy service account
kubectl create -n $NAMESPACE -f prp-htcondor-portal/provisioner/k8s/prp-htcondor-provisioner-account.yaml

2) create shared secret
kubectl create secret generic osg-wn-secret -n $NAMESPACE --from-file=prp-wn.token=prp-wn-${NAMESPACE}.token

3) create config file
kubectl create configmap osg-provisioner-config -n $NAMESPACE --from-file=osg_provisioner.conf=osg_provisioner.conf
